# RoboVerse

> 统一平台、数据集与基准，用于可扩展且可泛化的机器人学习。

## 关键数据

- **GitHub**: https://github.com/RoboVerseOrg/RoboVerse
- **星标**: 1737 ⭐
- **主要语言**: Python
- **最近更新**: 2026-05-10 (活跃)
- **组织**: RoboVerseOrg

## 核心特点

- **统一平台**: 整合仿真、数据收集、训练和评估
- **可扩展性**: 支持多种仿真器后端 (MuJoCo, SAPIEN, Isaac)
- **可泛化**: 跨场景、跨任务、跨机器人的测试
- **数据集**: 提供大规模机器人数据集

## 与其他实体的关系

- 支持多种仿真器后端，包括 [mujoco](mujoco.md)、[sapien](sapien.md)、[nvidia-isaac](nvidia-isaac.md)
- 定位为 [open-x-embodiment](open-x-embodiment.md) 的补充/竞品
- 与 [lerobot](lerobot.md) 的理念一致：提供端到端机器人学习工具链

# RoboTwin

> RoboTwin 2.0 官方仓库。双臂操作基准与数字孪生仿真平台。

## 关键数据

- **GitHub**: https://github.com/RoboTwin-Platform/RoboTwin
- **星标**: 2299 ⭐
- **主要语言**: Python
- **最近更新**: 2026-05-10 (活跃)
- **组织**: RoboTwin-Platform

## 核心特点

- **双臂操作**: 专注于双臂协作操作任务
- **数字孪生**: 支持真实世界与仿真之间的数字孪生对应
- **数据收集**: 支持从真实双臂机器人收集数据
- **2.0 版本**: 大幅扩展任务和环境

## 与其他实体的关系

- 与 [robosuite](robosuite.md)、[maniskill](maniskill.md) 一起构成操作基准三巨头
- 特殊在于双臂任务，与单臂基准形成差异化
- 数字孪生能力支持 [sim-to-real](../concepts/sim-to-real.md) 研究

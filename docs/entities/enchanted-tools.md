# Enchanted Tools

[官网](https://enchanted.tools) · [X/Twitter](https://x.com/enchanted_tools)

法国服务/社交场景机器人公司，产品 Miroki，Pepper/Nao 原班人马。

## Miroki 特点

- 身高 120-130cm / 体重 30-40kg / **轮式底座**（非双足）
- 头部表情显示屏，可呈现丰富情感
- 轻量级、软体材料包裹，适合与老人/儿童近距离互动
- 续航 6-8h，静音运行

## AI 架构

- **对话与情感交互**：LLM + 情感识别 + 语音合成
- **轮式 SLAM**：2D/3D 语义地图，电梯/门禁集成
- **操作有限**：主要递送小物品（药品、文件、水杯），预定义抓取策略

### 技术细节

- **轮式设计**：与双足人形不同，Miroki 采用轮式底盘，在室内环境中更稳定、能耗更低
- **头部情感显示**：通过可变表情的显示屏增强人机交互体验，这是其与其他机器人最大的区别化
- **轻量化设计**：软体材料包裹、30-40kg 体重，适合与老人/儿童近距离互动
- **静音运行**：续航 6-8h，运行噪音极低，不干扰医疗/养老环境

## 最新动态（2023-2026）

- **2023-2024**：在欧洲多家医院和养老机构开展试点，验证 Miroki 在医疗场景中的实际价值
- **2025 年中**：完成新一轮融资，扩大在欧洲医疗市场的部署
- **2025 年底**：发布 Miroki 更新版，增强了多语言支持和情感交互能力
- **2026 年初**：与多家欧洲医疗保健机构签订商业协议
- **商业教训**：深刻浸取了 SoftBank Pepper 的商业化失败教训，更侧重实际价值而非科技展示

## 竞争格局

- **细分市场定位**：与 [figure-ai](figure-ai.md) 、[tesla-optimus](tesla-optimus.md) 、[1x-technologies](1x-technologies.md) 等通用人形定位完全不同，专注医疗/服务/社交场景
- **轮式 vs 双足**：舍弃了双足运动的“高科技”光环，采用更实用的轮式设计，降低了成本和复杂度
- **情感交互壁垒**：头部情感显示屏是其核心区别化，其他公司很难快速复制
- **挑战**：医疗场景对安全性、可靠性、规章合规的要求极高，商业化周期较长

## 关键人物

- **Jérôme Monceaux**：CEO / 联合创始人，前 SoftBank Robotics Europe 总监，是 Pepper/Nao 团队的核心成员，深刻理解服务机器人的商业化挑战
- **团队背景**：多名核心成员来自原 SoftBank Robotics 团队，在服务机器人领域有丰富经验

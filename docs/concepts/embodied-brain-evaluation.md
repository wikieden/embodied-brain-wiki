# 具身大脑评估基准 (Embodied Brain Evaluation)

> 如何评估一个机器人是否"聪明"？不是只看它能不能完成任务，而是看它的记忆、推理、情感和学习能力。当前领域最大的空白之一——缺乏针对认知能力的标准化评估。

---

## 为什么现有基准不够

### 现有基准的聚焦

当前的机器人评估主要关注：

- **任务成功率:** 做完了没有？
- **时间效率:** 多快做完？
- **次数:** 尝试了几次才成功？

### 缺失的认知维度

但是，这些指标完全没有捕捉到：

- 机器人**是否记得**之前的经验
- 机器人**是否理解**失败的原因
- 机器人**是否能从错误中学习**
- 机器人**是否有自己的目标和动机**

**结果:** 一个只会“背下所有答案”的机器人，和一个真正“理解”任务的机器人，在现有基准上可能得到同样的分数。

---

## 五大认知能力的评估框架

### 1. 记忆评估 (Memory Evaluation)

**核心问题:** 机器人能否正确地记住、检索和使用信息？

#### 建议指标

| 指标 | 定义 | 测试方法 |
|------|------|---------|
| **情节回忆准确率** | 能否正确回忆某个时间点发生的事件 | 在长程任务中插入问题“之前 X 时你看到了什么？” |
| **语义记忆一致性** | 对同一事实的多次描述是否一致 | 在不同时间问同一个知识问题，看答案是否冲突 |
| **工作记忆容量** | 能同时跟踪多少个任务状态 | 逐步增加同时进行的子任务数量 |
| **记忆耐久性** | 信息保存多久不丧失 | 完成任务后几小时/几天后测试相关知识 |
| **主动回忆率** | 能否在需要时主动调用过去经验 | 设置需要借鉴过去经验的情境，看是否自发检索 |

#### 实现框架

- **记忆问答基准:** 在机器人任务轨迹中插入专门的记忆测试问题
- **对比基线:** 有记忆 vs 无记忆版本的表现差异
- **忘记曲线:** 随时间推移测量记忆衰减

---

### 2. 推理评估 (Reasoning Evaluation)

**核心问题:** 机器人能否进行因果推理、反事实思维和问题解决？

#### 建议指标

| 指标 | 定义 | 测试方法 |
|------|------|---------|
| **因果发现率** | 能否从观察中发现原因结果关系 | 多次实验后问"为什么 A 导致 B？” |
| **反事实准确率** | 对“如果当时...”类问题的回答质量 | 人为制造失败，然后问"如果当时力气大一点会怎样？” |
| **规划有效性** | 生成的规划在物理上是否可行 | LLM 生成规划 → 物理引擎验证 |
| **失败分析深度** | 失败后是否能正确识别根因 | 记录失败后的自我分析，与真实原因对比 |
| **推理时间** | 完成推理所需时间 | 测量从接受问题到产生答案的延迟 |

#### 实现框架

- **因果任务集:** 特殊设计的需要因果理解的任务
- **反事实挑战:** 主动制造“如果当时...”场景
- **物理基定检查:** 所有推理结果必须经过物理验证

---

### 3. 情感评估 (Emotion Evaluation)

**核心问题:** 机器人能否正确地感知、表达和使用情感？

#### 建议指标

| 指标 | 定义 | 测试方法 |
|------|------|---------|
| **情感识别准确率** | 能否正确识别人类情绪状态 | 展示含有明确情绪的场景，让机器人判断 |
| **情感响应适当性** | 对某情绪的反应是否合理 | 模拟危险、悲伤、开心等情景，看行为是否合适 |
| **情感表达一致性** | 内部状态与外部表现是否一致 | 内部情感标签 vs 行为/语言标签的一致性 |
| **共情评分** | 用户感受到的共情程度 | 人类评价者对机器人反应的共情评分 |
| **情感调节效果** | 情感是否改善了行为表现 | 有情感系统 vs 无情感系统的任务表现对比 |

#### 实现框架

- **情感场景库:** 标注了情绪状态的交互视频/音频/行为数据
- **人类评价:** 情感是主观的，必须有人类评价
- **行为对比:** 有/无情感模块的 A/B 测试

---

### 4. 自我意识评估 (Self-Awareness Evaluation)

**核心问题:** 机器人能否建立并维护自我模型？

#### 建议指标

| 指标 | 定义 | 测试方法 |
|------|------|---------|
| **身体图式准确性** | 对自身形态的理解是否准确 | 问"你的右手能够到达哪些位置？” |
| **能力边界识别** | 能否识别自己能/不能做什么 | 让它判断任务是否在能力范围内 |
| **动作意图检测** | 能否区分自己的动作和外部变化 | 被人推一下 vs 自己失去平衡，看否能区分 |
| **自我检测率** | 能否在适当时机检查自身状态 | 观察是否主动报告"我的电量低了”等状态 |
| **反思能力** | 能否评价自己的行为 | 失败后是否能分析"我为什么错了” |

#### 实现框架

- **镜像测试:** 让机器人在镜子/摄像头中识别自己
- **能力问卷:** 标准化的能力自评问题
- **自我监控:** 记录自我监测的频率和准确性

---

### 5. 终身学习评估 (Lifelong Learning Evaluation)

**核心问题:** 机器人能否在不忘记旧知识的同时学习新知识？

#### 建议指标

| 指标 | 定义 | 测试方法 |
|------|------|---------|
| **遗忘率** | 学习新任务后旧任务的性能下降幅度 | LIBERO 基准的标准流程 |
| **学习速度** | 掌握新任务所需的样本数/时间 | 在新任务上达到目标性能所需的交互次数 |
| **迁移能力** | 旧知识对新任务的帮助程度 | 测量新任务的学习速度 vs 零基线对比 |
| **在线适应性** | 在运行中学习的能力 | 不停机情况下的学习效率 |
| **技能巩固率** | 学会的技能是否稳定持久 | 在不同时间间隔后测试旧技能的稳定性 |

#### 实现框架

- **LIBERO:** 已有的终身学习基准
- **任务流:** 连续的任务流，模拟真实世界的不断变化
- **快照测试:** 在不同时间点对所有已学任务进行快照测试

## 综合评估框架: 认知能力成熟度 (Cognitive Capability Maturity)

### 五级成熟度模型

| 等级 | 名称 | 记忆 | 推理 | 情感 | 自我 | 学习 |
|-----|------|------|------|------|------|------|
| **L1** | 反射型 | 无 | 无 | 无 | 无 | 无 |
| **L2** | 记忆型 | 有 | 无 | 无 | 无 | 无 |
| **L3** | 推理型 | 有 | 有 | 无 | 无 | 无 |
| **L4** | 共情型 | 有 | 有 | 有 | 有 | 无 |
| **L5** | 自我进化型 | 有 | 有 | 有 | 有 | 有 |

**使用方法:** 对每个机器人/系统，在每个维度上进行 L1-L5 评分，形成认知能力矩阵。

### 当前产品的定位

- **大多数商业机器人:** L1 反射型
- **MemoryVLA / CogACT:** L2 记忆型
- **CSR / MolmoAct2:** L2-L3 边界
- **N.E.KO:** L4 共惄型（情感+记忆）
- **理想的具身大脑:** L5 自我进化型

# 具身大脑框架对比

> 对比当前主流的具身大脑相关开源框架，分析它们在记忆、推理、情感、学习上的架构选择。

---

## 总览

| 框架 | 主要能力 | 记忆架构 | 推理方式 | 情感/动机 | 学习能力 | 企业/机构 |
|------|----------|----------|----------|-----------|-----------|------------|
| **MemoryVLA** | 感知-认知记忆 | 双系统（感知+VLA 、认知记忆） | 无独立推理 | 无 | 训练后固定 | 学术界 |
| **CogACT** | 认知-动作协同 | 外部知识库 | LLM 引导分解 | 无 | 训练后固定 | Microsoft |
| **N.E.KO** | 具身情感 | 简单上下文 | 规则基策略 | 情感引擎 | 手动配置 | 开源社区 |
| **ReconVLA** | 重建式感知 | 内部潜在表征 | 通过重建验证理解 | 无 | 训练后固定 | OpenHelix |
| **CSR** | 实时认知引擎 | 巨量缓存状态表征 | LLM 作为认知引擎 | 无 | 在线更新 | 学术界 |
| **MolmoAct2** | 动作推理 | 简单上下文 | 动作前思考 | 无 | 训练后固定 | 开源 |

---

## 详细分析

### MemoryVLA

**架构:** 在标准 VLA 之上叠加了一个感知-认知记忆模块。

- **感知层:** 标准 VLA 处理当前观察
- **记忆层:** 将历史观察嵌入存储，支持类似经验检索
- **融合:** 当前观察 + 相关记忆 → 动作

**评价:** 进步在于给 VLA 增加了"记忆"，但记忆本身还比较简单，没有空间/时间结构化的组织。缺乏独立的推理和情感层。

### CogACT

**架构:** 明确分离了"认知"和"动作"两个层次。

- **认知层:** 先理解任务目标和环境状态
- **动作层:** 在认知的指导下执行具体动作
- **协同:** 认知层可以在动作前进行检查和调整

**评价:** 有意义的分层，但认知层依然比较简单，主要是任务分解，没有深层次的因果推理或反事实思维。

### N.E.KO

**架构:** 聚焦于情感交互层。

- **情感引擎:** 独立的情感状态管理
- **共情模块:** 检测用户情绪并响应
- **行为表达:** 将情感映射到机器人的语言和动作

**评价:** 情感方向的有趣尝试，但整体架构较为简化，缺乏对记忆、推理和学习的深度整合。

### ReconVLA

**架构:** 通过重建来验证理解深度。

- **重建头:** 从观察中重建世界状态
- **验证:** 重建质量反映了感知理解的深度
- **控制:** 基于重建结果做出决策

**评价:** 提出了一个别出心裁的视角：理解的证据是重建能力。但重建本身很耗费计算资源，且没有解决记忆、推理等更广泛的认知问题。

### CSR

**架构:** 解决 LLM 作为认知引擎时的延迟问题。

- **巨量缓存:** 将历史状态表征存储在缓存中
- **快速检索:** 无需重新计算，直接检索相关历史
- **实时性:** 支持无限时域实时策略

**评价:** 在工程上是重要突破，解决了 LLM 做认知引擎时的核心瓶颈（延迟）。但依然是"给 LLM 加了一个笔记本"，而非真正的统一认知架构。

### MolmoAct2

**架构:** 在动作前进行推理。

- **推理模块:** 执行前先"思考"动作后果
- **部署优化:** 专注降低成本和延迟
- **实用导向:** 面向真实世界部署

**评价:** 实用的部署方向工作，但推理模块相对简单，主要是动作前的检查，而非深度的因果推理或反思。

---

## 共同缺陷

所有当前框架都有一个核心问题：**它们都是“给 VLA 加了某个模块”，而非“重新设计了认知架构”**。

具体来说：

1. **无统一记忆系统:** 没有实现工作/情节/语义/程序四种记忆的统一管理
2. **无深度推理:** 没有真正的因果推理、反事实思维或元学习
3. **情感层弱:** 只有 N.E.KO 探索了情感，但与其他能力缺乏整合
4. **学习能力有限:** 大多数是训练后固定，缺乏真正的终身可塑性
5. **评估缺口:** 没有针对记忆、推理、情感等认知能力的标准化评估

---

## 理想中的具身大脑架构

综合以上分析，理想的具身大脑应该包含：

| 层次 | 功能 | 关键技术 |
|------|------|---------|
| **感知层** | 处理原始观察 | VLA / 感知器 |
| **世界模型** | 预测未来状态 | 视频生成 / 潜在动力学 |
| **记忆系统** | 统一管理四种记忆 | 向量库 + 回忆增强 |
| **推理引擎** | 因果推理、反事实、规划 | LLM + 世界模型验证 |
| **情感引擎** | 情感状态管理、内生动机 | 情感因果模型 |
| **自我模型** | 身体图式、能力边界 | 本体感受 + 反思 |
| **学习系统** | 终身学习、技能巩固 | 回放 + 正则化 + 元学习 |
| **行动层** | 执行低层动作 | VLA / Diffusion Policy |

**核心设计原则:** 
- 各层次之间有反馈循环（推理影响情感，情感调节学习率，学习改变记忆策略...）
- 每个层次都可以被独立评估和替换
- 支持在线更新，不需要停机重训
